Web application news and media from api The Guardian Open platform

### `include`

- React
- React-hook
- typescript
- styled-components

### `dependencies`

- craco
- axios
- classnames
- moment
- moment-timezone
- node-sass
- react-html-parser
- react-router-dom
- react-query

# Get start

## Install

```sh
yarn
```

## Run

```sh
yarn start
```
